## **ğŸ“¦ Cara Memainkan**

### **Opsi 1: Menggunakan File Executable (.exe)**
Cara termudah bermain tanpa perlu instalasi apapun!
1.  Buka folder `dist`.
2.  Jalankan file **`Word Games.exe`**.
3.  Selamat bermain!

### **Opsi 2: Menjalankan dari Source Code (Python)**
Jika ingin memodifikasi atau melihat kode programnya:
1.  Pastikan **Python 3.x** sudah terinstal.
2.  Install library yang dibutuhkan:
    ```bash
    pip install pygame
    ```
3.  Jalankan game:
    ```bash
    python main.py
    ```

## **ğŸ® Kontrol Permainan**

- **Klik Kiri (Drag):** Menandai kata di dalam kotak permainan.
- **Tombol HINT:** Membuka huruf awal dari kata yang belum ditemukan.
- **Tombol ESC:** Kembali ke menu sebelumnya (Saat bermain) atau Keluar (Di Menu Utama).
- **Tombol R:** Ulangi level saat waktu habis.
- **Tombol SPASI:** Lanjut ke level berikutnya saat menang.
